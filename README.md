## UI/UX Internship Task 2

# CRM Dashboard Design

## Project Overview
I designed a CRM Dashboard focused on customer management, business analytics, and efficient workflow tracking. The dashboard provides users with quick access to key metrics and customer information through a clean and modern interface.

## Design Goal
The goal of this project was to create an intuitive dashboard that helps users monitor business performance, manage customer data, and make informed decisions quickly.

## Pages Designed
- Login Page
- Register Page
- CRM Dashboard
- Prototype Flow

## Key UX Decisions
- Clean and organized dashboard layout
- Easy access to customer and sales data
- User-friendly navigation system
- Consistent design components
- Responsive and modern interface

## Color & Typography
- Used professional and modern color palette
- Clear typography for better readability
- Consistent visual hierarchy across screens

## Tools Used
- Figma
- UI Kit Components
- Icons & Design Assets

## Outcome
The final CRM Dashboard design improves usability, simplifies data management, and provides a smooth user experience for monitoring business activities.
